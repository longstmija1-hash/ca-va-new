# Ça Va — сайт-визитка

Одностраничный лендинг барбершопа + beauty в Екатеринбурге.

## Запуск локально

```bash
python3 -m http.server 5500
```

Откройте [http://localhost:5500](http://localhost:5500)

## Структура

- `index.html` — разметка
- `style.css` — стили
- `assets/` — видео и фотографии

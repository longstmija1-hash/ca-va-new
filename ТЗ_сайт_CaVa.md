## 2. Технологии

- **HTML + CSS + Vanilla JS** — никаких фреймворков, чистый код
- **Шрифты:** Google Fonts или Adobe Fonts — выбрать пару: один элегантный serif (для заголовков), один чистый sans-serif (для текста). Пример: Cormorant Garamond + Jost
- **Анимации:** CSS transitions + лёгкий scroll-reveal через IntersectionObserver (без библиотек)
- **Карта:** Яндекс.Карты embed (iframe)
- **Запись:** кнопки ведут на внешний YClients по ссылке
- **Адаптивность:** mobile-first, корректно отображается на телефоне, планшете, десктопе
- **Hero-видео:** фоновое видео в первом экране (без звука, `muted`, `playsinline`, `loop`) + обязательный fallback на изображение
- **Без CMS, без бэкенда** — статичный сайт, хостинг на GitHub Pages или Netlify

## 5. UX-требования

- Кнопка «Записаться» должна быть видна всегда — зафиксировать в шапке или добавить floating-кнопку на мобиле
- Телефонный номер кликабелен (`tel:`)
- Сайт должен грузиться быстро — фото сжать до WebP
- Видео в Hero должно быть оптимизировано: `mp4 (h264)` + `webm` (по возможности), без аудиодорожки
- Плавный скролл между секциями
- Навигация: логотип (якорь наверх) + 4 пункта меню: Услуги · Атмосфера · Массаж · Контакты

### Рекомендованные размеры медиа
- **Hero видео (desktop):** 1920×1080 (16:9), вес до 8–12 MB
- **Hero fallback image:** 1920×1080, `webp`, до 350–500 KB
- **Фото для блока «О нас»:** 1200×1500 (4:5), `webp`, до 250–400 KB
- **Фото для галереи:** 1200×1500 (4:5), `webp`, до 180–300 KB каждое
- **Минимум по качеству:** длинная сторона не меньше 1600 px для hero/fallback и 1200 px для остальных фото
